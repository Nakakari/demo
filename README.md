[//]: # (## Instalasi)

[//]: # (1. Clone repositori ini)

[//]: # (   ```)

[//]: # (    git clone https://github.com/Nakakari/app_perjalanan.git)

[//]: # (   ```)

[//]: # (2. Masuk pada folder)

[//]: # (    ```)

[//]: # (    cd app_perjalanan)

[//]: # (    ```)

[//]: # (3. Copy file .env.example ke .env)

[//]: # (    ```)

[//]: # (     cp .env.example .env)

[//]: # (    ```)

[//]: # (4. Install semua package yang diperlukan)

[//]: # (    ```)

[//]: # (     composer install)

[//]: # (    ```)

[//]: # (    Apabila terdapat error dalam menjalankan ini, dapat menggunakan alternatif berikut:)

[//]: # (    ```)

[//]: # (    composer install --ignore-platform-reqs )

[//]: # (    ```)

[//]: # (5. Install NPM)

[//]: # (    ```)

[//]: # (    npm install)

[//]: # (    ```)

[//]: # (5. Buat database dengan nama database yang diinginkan, misal 'perjalanan'.)

[//]: # (6. Sesuaikan konfigurasi database di .env)

[//]: # (    Silakan buka file .env pada IDE Anda, kemudian cek kodingan berikut:)

[//]: # (    ```)

[//]: # (    DB_CONNECTION=mysql)

[//]: # (    DB_HOST=127.0.0.1)

[//]: # (    DB_PORT=3306)

[//]: # (    DB_DATABASE=laravel)

[//]: # (    DB_USERNAME=root)

[//]: # (    DB_PASSWORD=)

[//]: # (    ```)

[//]: # (    Ganti menjadi:)

[//]: # (    ```)

[//]: # (    DB_CONNECTION=mysql)

[//]: # (    DB_HOST=127.0.0.1)

[//]: # (    DB_PORT=3306)

[//]: # (    DB_DATABASE=perjalanan)

[//]: # (    DB_USERNAME=root)

[//]: # (    DB_PASSWORD=)

[//]: # (    ```)

[//]: # (7. Generate app key)

[//]: # (    ```)

[//]: # (     php artisan key:generate)

[//]: # (    ```)

[//]: # (8. Import database pada folder:)

[//]: # (    ```)

[//]: # (    __DATABASE)

[//]: # (    ```)

[//]: # (    Ke PHPMyAdmin Anda.)

[//]: # (9. Buka aplikasi melalui browser!)

[//]: # (    ```)

[//]: # (    php artisan serve)

[//]: # (    ```)
